About

This is a free otome game/dating sim produced by sakevisual. The art was done by tooaya, the coding was done by Thee Forsaken One, and writing was done by sake-bento. This game runs on the Ren'py engine. For a full list of credits, please visit http://www.sakevisual.com/realistair/staff.html

Basic Help

To advance through the story, left-click or press the space or enter keys. When at a menu, left-click to make a choice, or use the arrow keys to select a choice and enter to activate it.

Game Menu

To save the game, edit your preferences, or view your current stats, use the menu buttons on the bottom right of the screen or the right side of the screen.

Stats
Shows you your current in-game stats including the date and how much money you have.
Save
Allows you to save a game by clicking on a save slot.
Load
Allows you to load a game by clicking on a save slot.
Option
Changes the game preferences (options/configuration):
Fullscreen
Switches between fullscreen and windowed mode.
Transition
Controls the display of transitions between game screens.
Skip Option
Chooses between skipping messages that have been already seen (in any play through the game), and skipping all messages.
Skip After Choice
Controls if skipping stops upon reaching a menu.
BGM, Effect
Controls the volume of the music and sound effects. The further to the right these are, the louder the volume.
Text Speed
Controls the rate at which text displays. The further to the right this slider is, the faster the text will display. All the way to the right causes text to be shown instantly.
Auto-Text Speed
Controls automatic advance. The further to the left this slider is, the shorter the amount of time before the game advances. All the way to the right means text will never auto-forward.
Title
Returns to the main menu, ending the current game.
